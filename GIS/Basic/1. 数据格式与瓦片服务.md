## 1.2 WMS、WFS和WMTS服务

> [[GeoServer-Usage]]

地理信息服务标准

- WMS
web map service 网络地址服务
栅格数据 图像 png jpeg

```js

// leaflet
// WMSOptions
// demo
`
http://localhost:8080/geoserver/tutorial-boundarylines/wms
?service=WMS
&request=GetMap
&layers=tutorial-boundarylines%3Ane_110m_admin_0_boundary_lines_land
&styles=
&format=image%2Fpng
&transparent=true
&version=1.1.1
&width=256
&height=256
&srs=EPSG%3A3857
&bbox=-10018754.171394622,10018754.171394628,-5009377.085697311,15028131.257091928
`

/**
	service string 服务 (eg: WMS | ...)
	request string ? (eg: GetMap | ...)
	layers string 图层 (eg: tutorial-boundarylines:ne_110m_admin_0_boundary_lines_land)
	styles string 样式 (eg: ?)
	format string 输出格式 (eg: image/png | ...)
	transparent boolean 是否透明 (eg: true | false)
	version string 版本 (eg: 1.1.1 | ...)
	width number ? 
	height number ?
	srs ? (eg: EPSG:4326 | ...)
	bbox 区域 (eg: 94.60546875,3.396972656250001,128.373046875,19.27880859375)
	exception string ? (eg: application/vnd.ogc.se_inimage) 
*/

// leaflet 预设
export namespace CRS {
    const EPSG3395: CRS;
    const EPSG3857: CRS;
    const EPSG4326: CRS;
    const EPSG900913: CRS;
    const Earth: CRS;
    const Simple: CRS;
}


```

- WMTS
web map tile service 网络地图瓦片服务
瓦片图像

```js

`
http://localhost:8080/geoserver/tutorial/gwc/service/wmts
?layer=tutorial%3Abeijing_18
&style=
&tilematrixset=WebMercatorQuad
&Service=WMTS
&Request=GetTile
&Version=1.0.0
&Format=image%2Fpng
&TileMatrix=15
&TileCol=26979
&TileRow=12408
`

```
