### 齐次坐标

在n维坐标的基础上，增加一维，使非线性变换可以用线性方式表示

| 原始坐标      | 齐次坐标         |
| --------- | ------------ |
| (x, y)    | (x, y, w)    |
| (x, y, z) | (x, y, z, w) |

w≠0 & w≠1时，实际坐标为：(x/w, y/w) 或 (x/w, y/w, z/w)
w=0时，表示为向量


### 线性变换 vs 非线性变换

>变换是一个函数

线性变换必须满足：
- f(a + b) = f(a) + f(b)
- f(kx) = kf(x)

<font color="#ff0000">平移是非线性变换</font>
<font color="#ff0000">旋转、缩放是线性变换</font>


```js
// 平移变换

P1 = (1,1)
P2 = (2,2)
// 平移量为：
T = (2,2)

// 方式1：f(P1 + P2)
// 先把两个点相加
P1 + P2 = (1+2, 1+2) = (3,3)
// 再对结果进行平移
f(P1 + P2) = (3+2, 3+2) = (5,5)

// 方式2：f(P1) + f(P2)
// 分别对两个点进行平移，再将结果相加
f(P1) = (1+2, 1+2) = (3,3)
f(P2) = (2+2, 2+2) = (4,4)
f(P1) + f(P2) = (3+4, 3+4) = (7,7)

```


```js

// 2倍缩放变换
P1 = (1,1)
P2 = (2,2)

// 方式1：f(P1 + P2)
P1 + P2 = (1+2, 1+2) = (3,3)
f(P1 + P2) = (3*2, 3*2) = (6,6)

// 方式2：f(P1) + f(P2)
f(P1) = (1*2, 1*2) = (2,2)
f(P2) = (2*2, 2*2) = (4,4)
f(P1) + f(P2) = (2+4, 2+4) = (6,6)

```
