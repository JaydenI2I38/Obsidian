
存储限定符
1. attribute
	1. 顶点有关的数据
2. uniform
	1. 与所有顶点相关（全局
3. textures
	1. 纹理（数据序列
4. Varying
	1. 可变变量
	2. 用来顶点着色器给片段着色器传值


数据类型
- 矢量
	- vec2、vec3、vec4（具有2/3/4个浮点数元素的矢量
	- ivec2、ivec3、ivec4（具有2/3/4个整形元素的矢量
	- bvec2、bvec3、bvec4（具有2/3/4个布尔值元素的矢量
- 矩阵
	- mat2、mat3、mat4（2\*2、3\*3、4\*4的浮点数元素的矩阵（分别具有4、9、16个元素


顶点着色器的内置变量
- vec4 gl_Position - 顶点位置
	- 必须
- float gl_PointSize - 点的尺寸（像素数）
	- 可选
	- 默认为 1.0


片元着色器的内置变量
- vec4 gl_FragColor - 指定片元颜色（rgba）


API

gl.drawArrays(mode, first, count)
参数
- mode
	- 绘制方式
	- gl.POINTS
	- gl.LINES
	- gl.LINE_STRIP
	- ...
- first
	- 从哪个顶点开始绘制
- count
	- 绘制需要用到多少个顶点
